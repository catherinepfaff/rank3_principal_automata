--- PAGE 1 ---

By an $a$-graph we mean a graph in which each of (1)--(4) holds:
1) There is exactly one red vertex; all other vertices are purple.
2) There is exactly one red edge; all other edges are either black or purple.
   Edges colored red or purple are called "colored edges."
3) Multi-edges are allowed, but for each pair of vertices there is at most one
   colored edge connecting that pair.
4) Loops are allowed, but only black edges can be loops.

Define
\[
\mathcal{L}=\{x_1,\bar{x}_1,\ldots,x_5,\bar{x}_5\},\qquad \bar{\bar{x}}_j=x_j.
\]
Assume
\[
x_1<\bar{x}_1<\cdots<x_5<\bar{x}_5
\]
is the increasing order on $\mathcal{L}$.

A distinguished admissible ltt structure is a connected graph $G$ with:
- 5 pairs of vertices labeled $x_1,\bar{x}_1,\ldots,x_5,\bar{x}_5$;
- $x_1$ is red and all remaining vertices are purple;
- $x_r$ is connected by a red edge to $x_2$;
- $\{\bar{x}_1,x_2,\bar{x}_2,\ldots,x_5,\bar{x}_5\}$ is partitioned into
  three sets of size $3$, and vertices in each part are connected by purple edges;
- each pair $\{x_j,\bar{x}_j\}$ is connected by a black edge.

Two colored graphs $G_1,G_2$ with vertices labeled by $\mathcal{L}$ are
epi-isomorphic iff there is a bijection $\phi:\mathcal{L}\to\mathcal{L}$ such
that:
(i) for each $x\in\mathcal{L}$, the vertices labeled $x$ and $\phi(x)$ have
    the same color;
(ii) for each $x,y\in\mathcal{L}$ with $x\neq y$, the vertices labeled
     $\phi(x),\phi(y)$ are connected by an edge of color $C$ iff the vertices
     labeled $x,y$ are connected by an edge of color $C$.

An admissible ltt structure is any graph epi-isomorphic to a distinguished
admissible ltt structure.

*First write code to list all admissible ltt structures*
(these will give potential nodes of the automaton).

--- PAGE 2 ---

Place some (any) order on the admissible ltt structures.
Let $\mathrm{PLTT}$ denote this ordered list.

Step 1: Start with a particular ltt structure $G$.
- $\mathrm{GRAPHS}=\{G\}$;
- $\mathrm{graphs}_1=\{G\}$;
- $\mathrm{boxes}=\{\mathrm{graphs}_1\}$;
- layer 1 consists only of $G$, i.e. $\mathrm{GRAPHS}_1=\{G\}$.

[diagram]

Remark: $\mathrm{GRAPHS}$ will ultimately be the list of all nodes in the
automaton.
- each $\mathrm{graphs}_k$ will be a "box" of epi-isomorphic nodes;
- $\mathrm{boxes}$ will be the collection of these epi-isomorphism classes;
- before closing loops where nodes represent the same epi-isomorphism class of
  ltt structures, the automaton looks schematically like:
  $\mathrm{GRAPHS}_1$ (layer 1), $\mathrm{GRAPHS}_2$ (layer 2),
  $\mathrm{GRAPHS}_3$ (layer 3), etc.

--- PAGE 3 ---

Recursively (with respect to $k$), complete in $\mathrm{PLTT}$ order Steps 2--3
for each $G\in \mathrm{GRAPHS}_k$.

Step 2: Find the 4 arrows (terminal ltt structures out of $G$).
Let $k$ denote the layer of $G$, i.e. $G\in\mathrm{GRAPHS}_k$.
- Input the starting ltt structure $G$ (an admissible ltt structure):
  vertices labeled by $\mathcal{L}$, with vertex colors and edge colors.
- Let $d_r$ be the label of the red vertex.
- Let $d_p$ be the label of the purple endpoint of the red edge.
- There are exactly two vertices connected to the $d_p$-vertex by purple edges:
  1) let $d_1$ be the label of the first;
  2) let $d_2$ be the label of the second;
  with order inherited from $\mathcal{L}$.
- For each of $d_1,d_2$, there are two potential directed edges out of $G$
  (hence one potential terminal ltt structure for each such edge).

For $j\in\{1,2\}$:
(2a) edge label: $d_r \to d_j\bar{d}_r$.
     The terminal ltt structure $G_{j;a}$ has:
     - red vertex $d_j$ and red edge $\{d_r,\bar{d}_j\}$;
     - precisely these purple edges:
       - a purple edge $\{d_j,d'\}$ whenever $G$ has a colored edge
         $\{d_r,d'\}$;
       - a purple edge $\{d,d'\}$ whenever $G$ has a purple edge $\{d,d'\}$ and
         neither $d=d_r$ nor $d'=d_r$.

(2b) edge label: $d_j \to d_r\bar{d}_j$.
     The terminal ltt structure $G_{j;b}$ has:
     - red vertex $d_j$ and red edge $\{d_j,\bar{d}_r\}$;
     - precisely these purple edges:
       - a purple edge $\{d_r,d'\}$ whenever $G$ has a colored edge
         $\{d_j,d'\}$;
       - a purple edge $\{d,d'\}$ whenever $G$ has a purple edge $\{d,d'\}$ and
         neither $d=d_r$ nor $d'=d_r$.

Let
\[
\mathrm{terminal}(G):=\{G_{1;a},G_{1;b},G_{2;a},G_{2;b}\}
\]
(repetitions allowed).
- Remove any graph from $\mathrm{terminal}(G)$ that is not in $\mathrm{PLTT}$.
- Each graph in $\mathrm{terminal}(G)$ is in level $k+1$.
- $\mathrm{terminal}(G)$ and level $k+1$ both inherit their order from
  $\mathrm{PLTT}$.

--- PAGE 4 ---

Step 3:
In $\mathrm{PLTT}$ order, apply the following to each $G'\in\mathrm{terminal}(G)$.

- If $G'$ is epi-isomorphic to some $G''\in\mathrm{GRAPHS}$, but $G'\neq G''$:
  - $\mathrm{GRAPHS}\leftarrow \mathrm{GRAPHS}\cup\{G'\}$;
  - $\mathrm{graphs}_j\leftarrow \mathrm{graphs}_j\cup\{G'\}$ where
    $G''\in\mathrm{graphs}_j$;
  - add a "permutation arrow" $G'\to G''$ labeled by $\phi$, where $\phi$ is
    the permutation of $\mathcal{L}$ from the epi-isomorphism.

- If there exists $G''\in\mathrm{GRAPHS}$ with $G'=G''$:
  add an arrow $G'\to G''$ with map $G'\mapsto G''$.

- For the remaining $G'$:
  - $\mathrm{GRAPHS}\leftarrow \mathrm{GRAPHS}\cup\{G'\}$;
  - $\mathrm{graphs}_{i+1}=\{G'\}$ where $\mathrm{graphs}_i\in\mathrm{boxes}$
    and $\mathrm{graphs}_{i+1}\notin\mathrm{boxes}$;
  - $\mathrm{GRAPHS}_{k+1}\leftarrow \mathrm{GRAPHS}_{k+1}\cup\{G'\}$.

Step 4:
After Steps 2--3 are completed for each $G\in\mathrm{GRAPHS}_k$, check whether
$\mathrm{GRAPHS}_{k+1}=\varnothing$.
If $\mathrm{GRAPHS}_{k+1}=\varnothing$, terminate.
If $\mathrm{GRAPHS}_{k+1}\neq\varnothing$, continue Steps 2--3 for each
$G\in\mathrm{GRAPHS}_{k+1}$.

Step 5:
Output the automaton as nodes and edges (including ltt structure/map info).
But what we really want is the strongly connected components.
